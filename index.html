Installing

Installing Not Enough Blocks is no different from other Forge mods

  • MultiMC
  • Minecraft Launcher
  • Developer
MultiMC is a free, open source launcher for Minecraft, managed by @peterixxx and @02JanDal. It allows you to have multiple, separate instances of Minecraft (each with their own mods, texture packs, saves, etc) and helps you manage them and their associated options with a simple interface.
- From the MultiMC website
MultiMC

Set up

  1. Create a new instance in MultiMC. Select either Minecraft version 1.8.9 or 1.7.10
  2. Right click the instance and hit Edit Instance, go to the Version tab and click Install Forge, select the latest version for the version of Minecraft you chose. It will now download and install Forge for you. Close the Edit Instance window when it's done.
  3. Right click the instance and hit Instance Folder. Open the minecraft folder and go to the mods folder. Put notenoughblocks-x.x.x-x.x.x.jar and it's dependencies from your downloads to this folder. Close the folder when you're done.
  4. Double click the instance to run Minecraft - you need to do this before you add any resource packs. When Minecraft has finished loading, close it.
  5. Open the Instance Folder again, go to minecraft>config>notenoughblocks>resourcepacks and put any resource pack you'd like in there.
  6. That's it, launch the game and enjoy your new blocks.
