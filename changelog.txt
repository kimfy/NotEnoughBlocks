+ Recipes! See changelog how to apply

+ Added changelog.txt

Recipes can either be a String, or array
Shaped recipe:
"recipe": [
    [ "crafting_table", "crafting_table", null ],
    [ null, "stick", null ],
    [ null, "stick", null ]
]

Shapeless recipe:
"recipe": "stick"
or
"recipe": [
    [ "stick", "crafting_table", "stick", "command_block" ]
]
